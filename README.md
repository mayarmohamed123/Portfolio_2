# Daniels - Portfolio Template

[![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://portfolio-2-xhgd.vercel.app/)

A sleek, professional, and fully responsive personal portfolio website built for designers and developers. This template features a modern aesthetic with smooth scrolling, project filtering, and a clean user interface.

## 🚀 Live Demo
View the live project here: [https://portfolio-2-xhgd.vercel.app/](https://portfolio-2-xhgd.vercel.app/)

## ✨ Features
- **Dynamic Typing Effect**: Engaging hero section with an animated introduction.
- **About Me Section**: Showcases professional summary and skill progress bars.
- **Service Offerings**: Highlighting core expertise like Web Design, Branding, and Development.
- **Filterable Portfolio**: Categorized showcase of works (Brand, Design, Graphic).
- **Client Testimonials**: Interactive carousel for customer feedback.
- **Contact Form**: Ready-to-use contact section with social media integration.
- **Fully Responsive**: Optimized for all devices using Bootstrap 5.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling and layout.
- **Bootstrap 5**: Responsive framework and components.
- **FontAwesome**: High-quality icons.
- **JavaScript**: Interactive elements and animations.

## 📁 Project Structure
```text
Portfolio_2/
├── CSS/               # Stylesheets (Bootstrap, Custom styles)
├── image/             # Portfolio and profile images
├── js/                # JavaScript files
├── webfonts/          # Font files for FontAwesome
└── index.html         # Main entry point
```

## 🛠️ Setup & Installation
1. **Clone the repository** (if you've added it to one):
   ```bash
   git clone <your-repository-url>
   ```
2. **Open the project**:
   Simply open `index.html` in your favorite web browser to view the site locally.

This project is optimized for deployment on platforms like **Vercel**, **Netlify**, or **GitHub Pages**.
